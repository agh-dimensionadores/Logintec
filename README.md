# Logintec Web – Rediseño Moderno

Sitio web renovado para [Logintec](https://www.logintec.com/) con diseño moderno que equilibra **logística crítica** y **equipos Cubiscan** al 50%.

## Características

- Hero dividido 50/50: logística a la izquierda, Cubiscan a la derecha
- Paleta de colores oficial Logintec: **borgoña** (`#5a1e31`) + **verde lima** (`#76b82a`)
- Diseño responsive (móvil, tablet, desktop)
- Animaciones suaves al hacer scroll
- Formulario de contacto integrado con email
- Sin dependencias — HTML, CSS y JavaScript puro

## Cómo ver el sitio

Abra `index.html` directamente en el navegador, o use un servidor local:

```bash
# Con Python
python -m http.server 8080

# Con Node (si tiene npx)
npx serve .
```

Luego visite `http://localhost:8080`
